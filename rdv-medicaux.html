Description du Projet

Une solution logicielle complète conçue pour fluidifier le parcours patient. Ce système interconnecte intelligemment l'administrateur, les médecins, le secrétariat et les patients pour une gestion hospitalière sans faille.

  • Technologies : PHP (Laravel), Vue.js, MySQL.
  • Fonctionnalités : Notification en temps réel, agendas dynamiques, gestion des annulations.
