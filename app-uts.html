Innovation Mobile

Cette application mobile vise à offrir une interface intuitive pour découvrir l'Université Thomas SANKARA (UTS), ses facultés et ses services essentiels directement depuis un smartphone.

Stack Technique

  • Framework : Flutter (Dart)
  • Architecture : Clean Architecture / Provider
  • Hébergement : Firebase Hosting

Ce que l'application propose

  • Présentation interactive des différentes filières de l'UTS.
  • Géolocalisation des bâtiments clés du campus universitaire.(Perspective)
  • Accès rapide aux actualités et communiqués officiels.
  • Mode hors-ligne pour consulter les informations essentielles sans connexion.(Perspective)
